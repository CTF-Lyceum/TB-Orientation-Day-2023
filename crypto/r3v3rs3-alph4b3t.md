# r3v3rs3-alph4b3t

## Description:
```
Кто-то перевернул английский алфавит в тексте...
XGUOBX{HFYHG1GFG3}
```

## Solution:
Чувствуем намек на переворачивание алфавита. Декодим substitute (он же шифр подстановок) с первернутым алфавитом и получаем флаг

### Flags: 
CTFLYC{SUBST1TUT3}
